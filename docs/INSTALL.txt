collective.sitter Installation
------------------------------

To install collective.sitter using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``collective.sitter`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        collective.sitter

* Re-run buildout, e.g. with:

    $ ./bin/buildout
